# Durable Chat App

![Aliasist banner](https://raw.githubusercontent.com/aliasist/aliasistabductor/master/images/aliasist_banner_orbit.png)

Aliasist uses this repo as a real-time chat template.

It provides:

- a small chat room experience
- durable shared messaging
- a Cloudflare Worker base
- a simple reference app for communication flows

This repo stays focused on chat and room-based messaging.
